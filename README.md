# Planning Toolkit

规划工具箱 — 一组用于任务规划与计划审查的 Claude Code Skills。

## Skills

| Skill | Description |
|-------|-------------|
| `task-planner` | 任务规划与执行引导，帮助把模糊目标变成清晰的行动步骤 |
| `plan-review` | 计划审查，从可行性、风险、完整性等维度系统审查计划 |

## Install

安装所有 skills：

```
npx skills add Jiangrb/task-planner
```

安装单个 skill：

```
npx skills add Jiangrb/task-planner --skill task-planner
npx skills add Jiangrb/task-planner --skill plan-review
```

安装后重启 Claude Code 即可使用。
